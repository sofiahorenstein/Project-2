Paw-Suit of Happiness

Pawsuit of Happiness is an analysis of pet data from the petfinder.com api as well as petfinder.com shelter data across the U.S. & Canada.

This analysis will include the following:

- A map marking down all shelter locations in the tri-state area featuring a drop down menu to filter by state!

- An interactive wheel visualization that provides photos and information for adoptable pets!

- Several visualizations for age, breed, and gender statistics for both adoptable dogs and cats!
